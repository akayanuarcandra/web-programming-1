Recommended Books

HTML

  • Introducing HTML5, by Bruce Lawson and Remy Sharp (New Riders)
  • HTML5, Up and Running, by Mark Pilgrim (O'Reilly Media and Google Press)
  • HTML5 Media, by Shelley Powers (O'Reilly)
  • The Definitive Guide to HTML5 Video, by Sylvia Pfeiffer (Apress)
  • HTML5 Canvas, by Steve Fulton and Jeff Fulton (O'Reilly)

CSS

  • Cascading Style Sheets: The Definitive Guide, by Eric Meyer (O'Reilly)
  • CSS3 for Web Designers, Dan Cederholm (A Book Apart)
  • CSS: The Missing Manual, by David Sawyer McFarland (O'Reilly)
  • CSS Cookbook: Quick Solutions to Common CSS Problems, by Christopher Schmitt (O'Reilly)
  • Handcrafted CSS: More Bulletproof Web Design, by Dan Cederholm (New Riders)

Scripting

  • Learning JavaScript, by Shelley Powers (O'Reilly)
  • JavaScript & jQuery: The Missing Manual, by David Sawyer McFarland (O'Reilly)
  • JavaScript: The Good Parts, by Douglas Crockford (O'Reilly)
  • DOM Scripting: Web Design with JavaScript and the Document Object Model, Second Edition by Jeremy Keith and Jeffrey Sambells (Friends of Ed)
  • JavaScript: The Definitive Guide, by David Flanagan (O'Reilly)

Mobile Web Design

  • Mobile First, by Luke Wroblewski (A Book Apart)
  • Responsive Web Design, by Ethan Marcott (A Book Apart)
  • Head First Mobile Web, by Lyza Danger Gardner and Jason Grigsby (O'Reilly)
  • Implementing Responsive Design: Building Sites for an Anywhere, Everywhere Web, by Tim Kadlec (Voices that Matter)

Web Design

  • Designing with Web Standards, by Jeffrey Zeldman (New Riders)
  • Adaptive Web Design: Crafting Rich Experiences with Progressive Enhancement, by Aaron Gustafson (Easy Readers)
  • Designing with Progressive Enhancement, by Todd Parker, Patty Toland, Scott Jehl, and Maggie Costello Wachs (New Riders)
  • Web Form Design, by Luke Wroblewski (Rosenfeld Media)
  • Ordering Disorder, Grid Principles for Web Design, by Khoi Vinh (Voices that Matter)

Accessibility

  • Pro HTML5 Accessibility, by Joshue O Connor (Professional Apress, 2012)
  • Universal Design for Web Applications: Web Applications that Reach Everyone, by Wendy Chisholm and Matt May (O'Reilly, 2008)

Site Performance

  • Even Faster Web Sites, by Steve Souders (O'Reilly Media)
  • High Performance Web Sites, by Steve Souders (O'Reilly)

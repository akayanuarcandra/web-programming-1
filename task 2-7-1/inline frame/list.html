The img element attribute List

src="url"
The location of the image file.
alt="text"
Alternative text.
width="number"
Width of the graphic.
height="number"
Height of the graphic.
usemap="usemap"
Indicates a client-side image map.
title="text"
Provides a "tool tip" when the user mouses over the image.Can be used for supplemental information about the image.
